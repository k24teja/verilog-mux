# Verilog Multiplexer Designs

## 📌 Overview
This repository contains Verilog implementations of multiplexers (MUX), which are fundamental components in digital design.

---

## 🧱 Modules Included
- 2:1 Multiplexer
- 4:1 Multiplexer

---

## ⚙️ Description

A multiplexer selects one input from multiple inputs based on select lines and forwards it to the output.
